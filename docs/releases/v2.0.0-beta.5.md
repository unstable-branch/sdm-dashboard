# v2.0.0-beta.5 candidate evidence

This prerelease supersedes the unpublished `v2.0.0-beta.4` publication attempt. Beta.4 passed the reviewed `dev -> main` application gates, but its tag workflow stopped during validation because the accelerator-contract script invoked `Rscript` before the workflow installed R. No beta.4 container images or GitHub Release artifacts were published.

## Correction

- The tag workflow now installs R before `pnpm run check:accelerators`.
- The release audit verifies that ordering so the same bootstrap failure cannot silently return.
- The beta.4 tag remains immutable; beta.5 identifies the corrected release commit.

## Carried candidate evidence

The application, deployment, accelerator, persistence, upgrade, rollback, and workflow evidence is unchanged from [`v2.0.0-beta.4.md`](v2.0.0-beta.4.md). Beta.5 changes release orchestration and public version metadata only.

## Publication gate

Publish beta.5 only after the tag workflow:

- validates tag/version/changelog/main ancestry;
- passes Node, Compose, accelerator, release, R smoke, R test, and release-audit gates;
- publishes all five immutable GHCR images;
- records all five image digests;
- produces source and Windows-ready archives plus `SHA256SUMS` and `image-digests.txt`;
- assembles a review-only draft GitHub prerelease for final inspection.
